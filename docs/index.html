VibeStudio

Run coding agents on your own machines. From anywhere.

VibeStudio is a desktop and iOS app for working with coding agents. It lets you view, edit, version, and run Agent Skills, and manage coding-agent sessions — like Claude Code — running on computers you own.

  • Your machines. Agents run on your own Mac or Linux computers, with your files, your tools, and your git history.
  • Skills, first-class. Browse, edit, version, and run Agent Skills without leaving the app.
  • From your pocket. The iOS app connects to your computers directly over SSH — no accounts, no relay in the middle.
